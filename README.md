## 🚀 Getting Started

CodeXFuel Pass is a **static web application** — no build step or backend required.

### Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/CodeXBlaze/CodexFuel.git

# 2. Open with any local server (e.g. VS Code Live Server, or Python)
cd CodexFuel

# Using Python 3
python -m http.server 8080

# Then open in your browser
# http://localhost:8080
```

Or simply open `index.html` directly in your browser — it works offline too.
